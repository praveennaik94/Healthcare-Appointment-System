# Healthcare Appointment Management System

## 📖 Overview

The Healthcare Appointment Management System is a full-stack web application that enables users to securely book, manage, cancel, and reschedule doctor appointments using JWT-based authentication. It provides a responsive React frontend and a robust Spring Boot backend with PostgreSQL integration.

---

## 🚀 Features

### 👤 Authentication

* User Registration
* User Login
* JWT Authentication
* Protected Routes
* Secure Logout

### 📊 Dashboard

* Total Appointments
* Booked Appointments
* Cancelled Appointments
* Quick Navigation Cards

### 📅 Appointment Management

* Book Appointment
* Dynamic Doctor Selection
* Dynamic Available Slot Loading
* Search Appointments
* Filter by Status
* Cancel Appointment
* Reschedule Appointment

### 👨‍⚕️ Doctor Management

* View Doctors
* Add Doctor
* Update Doctor
* Delete Doctor
* Show Only Available Doctors for Booking

### 🔔 Notifications

* Toast Notifications
* Kafka Integration (Backend)
